# SteadyPlan
A project management tool developed by Jonathan Wirick, Ryan Kovlaske, Daniel Losso-Kiss, and Kaleb Monti.
### Description
A web app for allowing users to create projects, manage tasks, collaborate with team members, and track progress with Kanban boards.
# Project Setup
1. Go to the official EnterpriseDB website download page for PostgreSQL (https://www.enterprisedb.com/downloads/postgres-postgresql-downloads) and download the latest version of PostgreSQL, correct for your OS
2. Run the install tool
3. Proceed with all default options, and don't forget the password set for your superuser
4. Launch pgAdmin 4, which you can find via your start menu
5. Click the dropdown that says 'Servers'
6. When prompted, enter the superuser password you set during instillation
7. Click the dropdown that says 'Postgres 1X'
8. Right-click on Databases -> Create -> Database
9. Name the Database steady_plan_db and click save
10. Right click on the new database, and press Query Tool
